# 의사코드

나중에 복잡한 코드를 제대로 설명하려면 지금부터 사소한 것이라도 계획, 순서를 작성하는 연습을 하자.

예시)
```
4개의 종이컵을 순서대로 뒤집어, 사탕을 찾기

의사코드
1. 종이컵의 개수만큼, 종이컵을 하나씩 뒤집습니다.
2. 뒤집은 종이컵에 사탕이 있다면,
3. 먹는다.

코딩
for(int cupNumber=1; cupNumber<=4; cupNumber++){ // 4개의 컵을 하나씩 확인하면서
	if (candy) { // 만약 사탕이 있다면
		eat(); // 먹는다
	}
}
```
